DevTools: Ferramentas Essenciais para Web

Se você trabalha com web, provavelmente já ouviu falar do DevTools. Mas sabe usar tudo que ele oferece?

O que é DevTools?

DevTools são ferramentas que vêm nos navegadores (Chrome, Firefox, Edge, etc.) para ajudar a ver e mexer nas páginas web.

Como abrir o DevTools?

  • Chrome, Edge e Firefox: Aperte F12 ou Ctrl + Shift + I (Windows/Linux) ou Cmd + Option + I (Mac).
  • Safari: Vá em Preferências → Avançado → Mostrar menu Desenvolvedor → Use Cmd + Option + I.

O que o DevTools faz?

  • Elements (Elementos): Veja e mude o HTML e CSS da página na hora.
  • Console: Veja erros, avisos e teste códigos JavaScript.
  • Network (Rede): Veja o que a página carrega e quanto tempo demora.
  • Performance: Veja como a página está rápida ou lenta.
  • Application (Aplicação): Mexa em dados da página (cookies, etc.).
  • Lighthouse: Testa se a página é boa em vários aspectos (velocidade, etc.).

Dicas Rápidas

  • Mudar texto na página: Clique duas vezes no texto e digite.
  • Ver como a página fica em celulares: Use Ctrl + Shift + M (no Chrome).
  • Achar mensagens no Console: Digite filter: para ver só o que interessa.
  • Copiar seletor CSS: Clique direito no elemento → Copiar → Copiar seletor.
